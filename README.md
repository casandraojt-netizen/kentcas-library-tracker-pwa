# Library Tracker PWA

Mobile Progressive Web App companion for Library Tracker desktop app.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to https://vercel.com and import the repository
3. Deploy — no environment variables needed
4. Open the deployed URL on your Android phone in Chrome
5. Tap the "Add to Home Screen" banner or use Chrome menu → Add to Home Screen

## First Launch

On first open, paste your Neon connection string (same one from your desktop app's `.env` file). Your library will load automatically.

## Features

- View and edit Physical Books and Web Collection
- Add/edit/delete books with full field support
- R18 blur, favorites, tags, progress bar
- Quick +1 chapter increment
- RSS new update badges
- Reading statistics
- Syncs with desktop app via shared Neon database
